Requirements:

Python 2.7 - download here: https://www.python.org/download/releases/2.7/
PIL        - to download, open terminal and enter $pip install PIL 
           - pip is required to make this installation

If you do not have both pip and python, it is recommended you run the EXE format instead.

To run:

Run the "Main.py" file with Python 2.7 in the "Program as a .PY" folder
 - or -
Run "Efficient Spell Calculator.exe" in the "Program as an EXE" folder
